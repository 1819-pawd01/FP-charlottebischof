HMTL Cheat Sheet

Element Name Element Description Element Example
<p>
Paragraph		 The <p> element specifies a paragraph of text and is an easy way to add text to your webpage. <p>Yesterday I created my first web page.</p>
<h1>
Heading 		The <h1> tag defines the most important heading. Smaller headings can be used after that one, for example, H2, H3, H4, ect. <h1>My Work Day</h1>
<br> The <br> element inserts a single line break. This tag is an empty tag which means that it has no end tag. Christmas <br>
Easter <br>
Fourth of July <br>
<body> The <body> element is used to house any information that is displayed in the main browser window. <body>
<h1> body text is displayed in here </h1>
<p> body text is displayed in here </p>
</body>
<footer> The <footer> contains information that is relevant to the body section. The footer is INSIDE of the body element. <footer>
<p> Copyright CMB 2018 </p>
</footer>
<b> The <b> element is used to BOLD words throughout your webpage. You can apply it to one word, or the whole sentence. <b> This morning I was very tired. </b>
OR
<b> This morning I was very tired. </b>
<i> The <i> element is used to italicize words on your webpage. Like the bold tag, it can also be applied to one word or a whole sentence. <i> This morning I was very tired. </i>
OR
<i> This morning I was very tired. </i>
<s> The <s> element is used to represent words that are no longer needed or relevant to the text that you published. <s> Thanksgiving is always on November 25th. </s>
<blockquote> The <blockquote> element indents a quote that you may use on your website, and when there is a screen reader, will most likely say "quote." <blockquote>
"That's one small step for a man, one giant leap for mankind."
</blockquote>

CSS Cheat Sheet

Property Name Property Values Property Description
background-color: (color); You can use RBG values, HEX values, or color names. The background-color; property is used to change the color of the background on your webpage.
font-size: (font size); The font-size; can be an absolute, or relative size. Absolute size sets the text to a specified size, and relative size sets the size relative to surrounding elements. The font-size; is used to change the size of the fonts. You can choose multiple different sizes depending on which sections you apply the font-size; propety to.
text-align: (alignment); You can align with left, right, center, or justify. The text-align; property is used to align differnt sections of your webpage.
font-variant: (variation); You can use this with either small caps or normal. The font-variant; property tells whether or not the text will be displayed in small-caps or normal.
text-color: (color); You can use a color name, a HEX value, or an RGB value. The text-color; property is used to change the color of the text.
text-decoration: (decoration); You can apply none, overline, underline, or line-through. The text-decoration; property is used to set or remove decorations from the text.
text-transform: (transform); You can apply capitalize, uppercase, or lowercase. The text-transform; property is used to specify uppercase or lowercase letters in your text.
letter-spacing: (space); You can use value, or length when spacing letters. The letter-spacing; property is used to specify the space that the characters have in the text.
text-indent: (indent); You can use the length value. The text-indent: property is used to specify the indentation of the first line on your webpage.
text-direction: (direction); You can use the ltr or rtl values. The text-direction property is used to change the direction of an element in HTML.

Attribute Cheat Sheet

Attribute Name Attribute Values Attribute Description Elements
<img src="source" alt=text /> The acceptable values are the URL or file name of the image. The attribute is used to bring a picture into your webpage. src means source, which is where your image is stored. This attribute can be added to the <img> element.
<img src="source" alt=text describing the attached image/> The acceptable values are the alt value. This attribute is used to describe the picture that is attached to your webpage. If the image is unable to display, this attribute will describe in words what it is. This attribute can be applied to the <img> element.
<a href= "url"> link text> The acceptable values are the URL of the website. The href attribute specifies the destination address of the link. This attribute can be applied to <a> or <link>.
<a href="url"> target="_blank" > The acceptable values are _blank or _self. The target attribute specifies where to open the document that you linked on your webpage. This attribute can be applied to <a>.
<a href="chick-fil-a.com"> target="_blank" title="Chick-Fil-A's Website!"> The acceptable values are title="_" whatever that name of the website is that you are linking. The title attribute gives extra information about the element. This attribute can be applied to <a> or <text>.
